What is Attribute?

HTML attributes are special words which provide additional information
about the elements or attributes are the modifier of the HTML element.
Each element or tag can have attributes, which defines the behaviour
of that element.Also, to make sure the entire element is always covered,
set the background-attachment property to fixed:This way, the background
image will cover the entire element, with no stretching (the image will keep
its original proportions)
